# HomeHero Android

A production-grade on-demand home services application built using modern Android development practices.

## Overview

HomeHero allows users to discover, book, and manage home services such as:

- Electrician
- Plumbing
- Cleaning
- AC Repair
- Carpenter
- Appliance Repair

The project is built as a learning platform for Android Lead Engineer practices including architecture, performance, testing, CI/CD, and production monitoring.
